# Power Apps FluentIcon Reference Guide

A visual reference for every **`FluentIcon`** value available in modern Power Apps canvas apps — **181 icons** with outline and filled previews.

Use this guide to pick the right icon name when building with **Modern Icon**, **Modern Button**, **Data Grid**, and other Fluent-based controls.

---

## What is FluentIcon?

Power Apps modern controls use Microsoft's **Fluent UI** icon set. Each icon is a typed enum value you reference in Power Fx as `FluentIcon.IconName`.

Example:

```powerfx
FluentIcon.Home
FluentIcon.CameraDome
FluentIcon.CheckmarkCircle
```

This is separate from the older **`Icon.*`** enum used by classic controls (like the legacy Icon control). Modern apps should use **`FluentIcon.*`**.

## Which controls use FluentIcon?

Common modern controls that accept `FluentIcon` values:

- **Modern Icon** — standalone icon display
- **Modern Button** — optional leading/trailing icon
- **Data Grid** — column icon properties
- Other Fluent-based modern controls with an **Icon** property

**Note** — the Toolbar control does not appear to accept the new FluentIcons icon set as of June 2026.

## How to set an icon

On a control with an **Icon** property, set the enum value directly:

```powerfx
Icon: =FluentIcon.Home
IconStyle: =IconStyle.Filled
```

You can also use the icon name as text on updated modern controls — both forms work:

```powerfx
Icon: ="Home"          // text
Icon: =FluentIcon.Home  // enum (recommended)
```

## Outline vs filled

Most modern icon controls have an **IconStyle** property:

| IconStyle | Look |
|-----------|------|
| `IconStyle.Outline` | Line / regular style |
| `IconStyle.Filled` | Solid / filled style |

The **Outline** and **Filled** columns in the table below show what each icon looks like in each style.

## Build an icon picker gallery

Create a collection from a table, then bind a gallery's **Modern Icon** to each row:

```powerfx
ClearCollect(
    colIcons,
    Table(
    { Icon: FluentIcon.Home, Label: "Home" },
    { Icon: FluentIcon.Search, Label: "Search" },
    { Icon: FluentIcon.Settings, Label: "Settings" },
    { Icon: FluentIcon.Person, Label: "Person" },
    { Icon: FluentIcon.Calendar, Label: "Calendar" },
    { Icon: FluentIcon.Mail, Label: "Mail" }
    )
);

// Gallery template — Modern Icon control:
//   Icon:       =ThisItem.Icon
//   IconStyle:  =IconStyle.Outline
// Optional label — Text label:
//   Text:       =ThisItem.Label
```

Add more rows using any **Name** from the table below. Each row's **Icon** column shows the exact Power Fx value to use.

## Good to know

- **181 icons** are in the `FluentIcon` enum — this guide lists all of them.
- This is **not** the full Fluent UI library (~2,900 icons). Only enum members work natively with `FluentIcon.*` in canvas apps.
- Preview images are reference SVGs from [Fluent UI System Icons](https://github.com/microsoft/fluentui-system-icons). They may look slightly different in your app depending on control version and theme.
- If an icon shows as a **circle placeholder** in Studio, the icon may not be available in your app's control version yet — try updating the app or picking a similar icon from the list.
